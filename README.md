## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/MusfirahSheikh8/Musfirah-Sheikh-Portfolio.git
```

Navigate to the project directory:

```bash
cd Musfirah-Sheikh-Portfolio
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open your browser and visit:

```bash
http://localhost:3001
```

---
